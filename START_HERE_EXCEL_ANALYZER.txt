FASTEST WAY TO RUN (Copy & Paste):
----------------------------------

Option 1 - Python Code:

  run_analytics()

Option 2 - Command Line:
  python -m features.excel_analyzer

Option 3 - Advanced (with custom output):

  run_analytics(output_file='my_analytics.xlsx')


WHAT YOU GET:
-----------
An Excel file (analytics_report.xlsx) with 3 sheets:

  Sheet 1: "Deals"
    - Client Name, Deal Status, Revenue, Source File

  Sheet 2: "Summary"
    - Total Deals, Wins, Losses, Win Rate %, Revenue, Avg Deal Value

  Sheet 3: "Top FAQs"
    - Top 5 FAQs that led to successful deals (with counts)

YOUR REPORTS FORMAT:
-------------------
Put text files in reports/ folder. Each report should have:

  Client: CompanyName (or "Company:", "Account:")
  Status: Win (or "Loss", "Success", "Failed")
  Revenue: $125,500 (or "$87300" or "Amount: 87300")

  Optional FAQs:
  FAQ 1: Your question here
  FAQ 2: Another question


ADD YOUR REAL DATA:
